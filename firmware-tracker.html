Engineering Assumptions & Notes
  • "Update Required" is auto-derived: YES if Current Firmware ≠ Approved Firmware (exact string match, case-insensitive).
  • No official firmware versions are pre-loaded. Approved firmware must be entered manually per device type and site requirements.
  • Always verify approved firmware against the manufacturer's latest service bulletin before performing any update.
  • Firmware updates on protection relays, SCADA gateways, and safety-critical devices must follow a formal change management and testing procedure.
  • Data is stored in browser session only and is not persisted. Export CSV before closing the page.
