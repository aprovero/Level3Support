Engineering Assumptions & Formulas
  • Adjusted Module Voc at $T_{min}$ = $Voc_{STC} \times [1 + \frac{\beta}{100} \times (T_{min} - 25)]$.
  • Adjusted Module Vmp at $T_{max}$ = $Vmp_{STC} \times [1 + \frac{\gamma}{100} \times (T_{max} - 25)]$.
  • String extreme Voc = Module adjusted Voc $\times$ Proposed Modules in Series.
  • String extreme Vmp = Module adjusted Vmp $\times$ Proposed Modules in Series.
  • Max modules by cold Voc = $\lfloor \text{Inverter Max DC Input Voltage} / \text{Module adjusted Voc} \rfloor$.
  • Min modules by hot Vmp = $\lceil \text{Inverter Min MPPT Voltage} / \text{Module adjusted Vmp} \rceil$.
  • Max modules by hot Vmp & MPPT Max = $\lfloor \text{Inverter Max MPPT Voltage} / \text{Module adjusted Vmp} \rfloor$.
  • STC cell temperature baseline is fixed at 25°C per international solar standard testing.
