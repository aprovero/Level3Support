Engineering Assumptions
  • Average Current (Iavg) = Σ(In) / N
  • Deviation % = (In − Iavg) / Iavg × 100
  • Status: NORMAL if |Deviation| ≤ Threshold; WATCH if Threshold < |Deviation| ≤ 2×Threshold; HIGH if |Deviation| > 2×Threshold or current < 0.
  • Overall: Normal if no flagged strings; Watch if any WATCH; Investigate if any HIGH.
  • Irradiance and module count are recorded for context only — no auto-correction to STC is performed.
