Engineering Assumptions
  • Reference Irradiance (GRef) default is 1000 W/m² (STC standard).
  • Uncorrected PR assumes STC module rating is constant across all operating module temperatures.
  • Temperature-corrected PR uses module/cell temperature coefficient to normalize rating against the STC reference of 25 °C.
  • PR status ranges: PR ≥ 80% is Normal, 75% ≤ PR < 80% is Watch (possible soilage or mild inverter curtailment), PR < 75% is Investigate.
