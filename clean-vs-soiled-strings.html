Engineering Assumptions & Limitations
  • All compared strings must be electrically identical (same model panels, tracker angles, inverter settings).
  • Values are mathematically normalized relative to reference module count if mismatch exists, though matching module count is recommended.
  • This comparison evaluates soiling but does not distinguish it from shading, blown fuses, bypass diode failures, or degradation. Pair with visual check.
