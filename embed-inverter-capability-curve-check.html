Engineering Assumptions
  • Formula: S_required = √(P² + Q²).
  • Utilization % = S_required / (S_rating × Thermal Derate × Voltage Factor × Inverters Online) × 100.
  • If operating point exceeds capability, the maximum active power limit for a fixed Q is computed as P_max = √(S_rating² - Q²), and the maximum Q for a fixed P is Q_max = √(S_rating² - P²).
